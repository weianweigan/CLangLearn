# 变量和常量

常量：在程序执行过程中值不会发生变化的量；

    整形常量 1，790
    实型常量 3.14，5.26，1.999
    字符常量:由单引号包围的单个字符或转义字符 如： ‘\t’（转义字符） '\015'(代表三位八进制数) ‘\x7f’（两位16进制数）
    字符串常量：由双引号包围一个或多个字符组成的序列，比如："fdsdff"
    标识常量:#define

变量：用来保存一些特定内容，并且在程序执行过程中值随时会发生变化的量
